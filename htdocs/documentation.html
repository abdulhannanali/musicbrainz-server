<& /comp/sidebar, title=>"MusicBrainz Information/Documentation", expand=>'documentation' &> We are the first to admit that the documentation of this project could be a bit better. We would love to have the time to work on this. But first we want to implement the best functionality for this project. Each of the following sections point to one page of documentation.

Introduction The general introduction to MusicBrainz gives an overview of the key features and content of MusicBrainz.
Moderation Introduction The introduction of the moderation process explains the inner working of MusicBrainz. MusicBrainz relies on the users to keep the database up to date.
Project Status The project status page give you an overall view of what we've accomplished and what still needs to get done.
Frequently Asked Questions The Frequently Asked Questions page answers general stuff and licensing questions.
To Do List For the short term, the To Do page lists the open tasks. If you would like to join us, this would be the place to look into for things we need help on.
Rob's To Do List Rob's To Do list shows you all the things that Rob should be working on.
How does it work? In the 'how does it work?' pages, the client/server model, music identification, playlist exchange, and MP3/Vorbis features are all explained.
Client HOWTO The Client HOWTO page gives you some documentation on how to use the MusicBrainz client library in your programs to access the MusicBrainz server.
The Future Our goals for the next years.
History You can read more about the roots of the project and a bit of project history in our historical department.
Biography Page Listed on the biography page are the people that worked on MusicBrainz. Please be warned that this page also contains pictures of the developers!

For Software Developers

For software developers wishing to develop third-party applications, we have these two libraries available:

  • The MusicBrainz Client Library / SDK provides low-level access to the MusicBrainz Server RDF interface and the TRM Signature Server.
  • The TunePimp Library / SDK provides complete functionality for writing your own MusicBrainz enabled tagging application.

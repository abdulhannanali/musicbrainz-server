Please note that we've moved our mailing lists (away from sourceforge, onto MusicBrainz itself).  Posting to the old addresses (@lists.sourceforge.net) will no longer work, so please update your address book entries with the new addresses, at lists.musicbrainz.org.  See below for details.

Our mailing lists are managed by Mailman, which means you can subscribe, unsubscribe, change your settings, and do all sorts of things either via the web or via e-mail.

Announcements

To hear about new MusicBrainz software, services, and other announcements, please subscribe to the musicbrainz-announce mailing list.

  • Subscribe to musicbrainz-announce
  • View the musicbrainz-announce archives

General Discussion

If you are interested in keeping in touch or getting involved with the MusicBrainz project, please subscribe to the musicbrainz-users mailing list.  This list is designed for general discussion about the MusicBrainz project and any general questions and issues that might arise.

  • Subscribe to musicbrainz-users
  • View the musicbrainz-users archives

Style

If you are interested in keeping up with the developement of the MusicBrainz Style Guidelines, please subscribe to the musicbrainz-style mailing list.

  • Subscribe to musicbrainz-style
  • View the musicbrainz-style archives

Internationalisation

The musicbrainz-i18n list is for the discussion of internationalisation (aka "i18n") issues within MusicBrainz.

  • Subscribe to musicbrainz-i18n
  • View the musicbrainz-i18n archives

Development

If you are interested in helping with or keeping track of MusicBrainz development, please subscribe to the musicbrainz-devel mailing list.

  • Subscribe to musicbrainz-devel
  • View the musicbrainz-devel archives

CVS Commits

The musicbrainz-commits mailing list is an automated notification of CVS commits.  Whenever someone checks in code into CVS for any of the MusicBrainz projects, a mail will be sent to this list.  This list is handy for keeping track of changes to the codebase.

  • Subscribe to musicbrainz-commits
  • View the musicbrainz-commits archives

Auto-Moderator Discussion

The musicbrainz-automods list is for discussion among automatic moderators.  Only they can post to this list, but anyone may subscribe to the list and read their postings.

  • Subscribe to musicbrainz-automods
  • View the musicbrainz-automods archives

If you are interested in keeping in touch or getting involved with the MusicBrainz project, please subscribe to the musicbrainz-users@lists.sourceforge.net mailing list. This list is designed for general discussion about the MusicBrainz project and any general questions and issues that might arise.

If you are interested in helping with or keeping track of MusicBrainz development, please subscribe to the musicbrainz-devel@lists.sourceforge.net mailing list.

The musicbrainz-commits@lists.sourceforge.net mailing list is an automated CVS commit mailing list. Whenever someone checks in code into CVS for any of the MusicBrainz projects, a mail will be sent to this list. This list is handy for keeping track of changes to the codebase.

  • Subscribe to the MusicBrainz users mailing list.
  • Subscribe to the MusicBrainz developers mailing list.
  • Subscribe to the MusicBrainz CVS commits mailing list.

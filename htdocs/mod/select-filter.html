What type of moderations do you want to view?

If you don't know, go for Vote on new moderations.  Please read the Style Guide, the Moderation Introduction and the Moderation FAQ before you start.

  • Vote on new moderations - lists open moderations which you haven't yet voted on, so you can vote on them.  If you don't know which one to choose, choose this one.
  • New moderations for your subscribed artists - as above, but for your subscribed artists only.
  • Vote on new imports from FreeDB - lists open "import from FreeDB" moderations which you haven't yet voted on, so you can vote on them.
  • Review your own moderations - lists moderations you entered into the system
  • Review your votes - lists moderations on which you've already voted
  • Review your failed moderations - lists moderations you entered into the system, but which were not accepted into the database
  • Advanced Moderation Search - lets you do all of the above, and more
  • Jump to a known moderation ID - only if you've got a particular mod ID to look up

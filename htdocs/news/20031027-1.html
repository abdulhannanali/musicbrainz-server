MusicBrainz Tagger 0.10.1

This version fixes a number of bugs from the 0.10.0 version.  Some MP3s would have corrupted ID3 tags after being tagged with the tagger - this is now fixed.  If you have any files that got corrupted by the tagger, download and install this new version, and then enable the 'clear tags before writing' option.  Then re-identify all of the corrupt files and write new tags.

Also, tagging Ogg/Vorbis files would crash fairly often, and tagging via proxy was also broken.  Both of these are now fixed.

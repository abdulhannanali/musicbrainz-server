Recent MusicBrainz News

2004-04-99 Server software updated - this release adds support for data replication, makes others' votes more visible, and allows moderation notes to be e-mailed to other voters, not just the moderator.  There are lots of other changes too.  read more...
2004-04-24 The 0.10.4 version of the MB Tagger has been released. This version fixes three minor bugs releated to saving tracks. read more...
2003-10-27 The 0.10.1 version of the MB Tagger has been released. This version fixes a number of bugs from the 0.10.0 release. read more...
2003-10-13 A new MusicBrainz Tagger has been released. After many months of writing new infrastructure and fixing bugs the 0.10.0 version should be stable and much more accurate than its predecessors. read more...
2003-06-24 libmusicbrainz 2.0.2 released. This version has Mac OS X support and a new build system.
2003-04-05 A new version of the MusicBrainz tagger has been released. Version 0.9.0 adds a few new features like watching directories, more powerful file renaming and improved proxy support. Tons of bugs have been fixed, and ID3 and Ogg Vorbis tag writing should be a lot more stable now. read more...
2003-03-29 New, faster database server!.  read more...
2003-03-08 libmusicbrainz 2.0.1 released. This version has minor bug fixes, improved python bindings and new perl bindings.
2003-02-18 Added Ruby bindings to the client download page. Thanks to Paul Duncan (pabs).
2003-02-11 The new MusicBrainz site layout was rolled out as part of the official MusicBrainz launch. The press release, the non-profit white paper and the new related artists feature are all part of the MusicBrainz launch.
2003-02-07 MB Tagger 0.8.5 has been released. This version adds a menu, fixes some minor bugs and makes the embedded browser behave better.
2003-02-07 Tom Heady, Jack Cummings, Sander von Zoest, Duane Harkness, Greg Junell have donated a total of $380 to MusicBrainz to pay for the upcoming press release.
2003-02-06 MB Tagger 0.8.4 has been released. Proxy support works as expected and other minor usability enhancements have been added.
2003-02-01 MB Tagger 0.8.3 for Windows has been released! This version support complete TRM collisions, similarity highlighting and a number of other important bug fixes.
2003-01-29 Sebastian Spaeth just donated $15!
2003-01-26 libmusicbrainz-2.0.0-pre4 has been released! This latest release supports the new MusicBrainz Metadata initiative version 2.1 and supports the FileLookup interface for writing MusicBrainz enabled tagger applications.
2003-01-10 MB Tagger 0.8.2 for Windows has been released! This version fixes a number of bugs and has basic support for resolving TRM collisions.
2003-01-06 Added an html version of the MusicBrainz license issues white paper.

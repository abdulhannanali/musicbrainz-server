Introduction

The CD Lookup tool is designed to allow users to lookup and contribute metadata for Audio CDs. The embedded web browser makes it easy to go through a number of Audio CDs and submit the metadata to the MusicBrainz server. This 32 bit Windows application runs on Windows 95, 98, Me, NT, 2000 and XP.

This tool is the predecessor to the MusicBrainz Tagger application. All of the features in the CD Lookup tool are also present in the Tagger application -- this tool is being provided for legacy purposes.

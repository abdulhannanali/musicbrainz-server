Introduction

The MusicBrainz client library (also referred to as mb_client and libmusicbrainz) is a development library geared towards developers who wish to add MusicBrainz lookup capabilities to their applications.

If you're looking for a complete, usable program to run, please try out these:

  • Use the MusicBrainz Tagger to automatically identify and label your digital music collection
  • Use the MusicBrainz CD Lookup Tool to update MusicBrainz from your CD collection

The client library is officially supported in 32 bit Windows (95, 98, Me, NT, 2000, XP), Linux and Mac OS X, with packages released for the RedHat and debian distributions. For more details on how to use this library, please take a look at the Client HOWTO.

The "Advanced Search" form on the Search page is actually the same form that the Tagger uses; it allows you to execute a combined search for artist, album and track all in one go.

How It Works

There is another significant difference between the main search form and this one.  The main search form does an exact word-by-word search (e.g. searching for "castle" will never find "castles"); it also shows all results which match your search criteria at all. 

On the other hand, the Advanced Search (Tagger) form sometimes uses a more flexible search based on the spelling of the words, not the whole words - so for example "hate" would now also match "hates", "hat" and "have" (with varying degrees of similarity).  At the same time, this type of search also takes into account the whole name of the track (album, etc) when judging how similar they are.

Example

If you search for the album "Boys" by "Tori Amos" (leaving the track name blank), then what's actually found is Toys, and NOT Boys for Pele.  What's happened here is that "Toys" is quite similar to "Boys" - only one letter is different - but "Boys for Pele" is quite different - there are two extra words!  Since there is a similarity threshold below which results are not shown, you end up finding one album but not the other.

Conclusion

When using the Advanced Search form, it might not be enough to enter one or two key words - for best results, enter the whole name of the artist, and album and/or track.

Introduction

The MusicBrainz server is a relational database plus a collection of web pages and web services providing access to that data.

The Database

The MusicBrainz database is built on the Postgres relational database engine.  Therefore these files are in Postgres "dump" format, only really suitable for restoring to a Postgres database.

Miscellaneous

  • MusicBrainz MySQL Data (tar.bz2) suitable for importing into MySQL.
  • MusicBrainz RDF Data dump (bz2) suitable for parsing with any RDF parser.
  • MusicBrainz Access DB (zip) MS Access database suitable for importing the above file into MS Access. [Details] (updated 2002-03-13)

Other data files are available from time to time on the MusicBrainz FTP Server.

Introduction

TunePimp is a library which can be built into other programs -- it is not a standalone end-user application.  TunePimp is the core of the MusicBrainz Tagger Application and compiles on all three major operating systems: Linux, Mac OS X and even Windows.

If you're looking for a complete, usable program to run, please try out these:

  • Use the MusicBrainz Tagger to automatically identify and label your digital music collection
  • Use the MusicBrainz CD Lookup Tool to update MusicBrainz from your CD collection

Language Bindings

Python and Perl bindings for the TunePimp library are included in the source distribution of TunePimp. Please check the python or perl subdirectories in the source distribution for more details.

Other releases are available on the MusicBrainz FTP Server.

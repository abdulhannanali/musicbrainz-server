Introduction

The TunePimp library (also referred to as libtunepimp) is a development library geared towards developers who wish to create MusicBrainz enabled tagging applications.

If you're looking for a complete, usable program to run, please try out these:

  • Use the MusicBrainz Tagger to automatically identify and label your digital music collection
  • Use the MusicBrainz CD Lookup Tool to update MusicBrainz from your CD collection

The TunePimp library is officially supported in 32 bit Windows (95, 98, Me, NT, 2000, XP), Mac OS X, and Linux, with packages released for the debian distribution. For more details on how to use this library, please take a look at the TunePimp Documentation.

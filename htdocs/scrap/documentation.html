Here is the documentation that we've written for MusicBrainz so far:
How does it work? In the 'how does it work?' pages, the client/server model, music identification, playlist exchange, and MP3/Vorbis features are all explained.
Client HOWTO The Client HOWTO page gives you some documentation on how to use the MusicBrainz client library in your programs to access the MusicBrainz server.
Client Library Reference The Client Library Reference contains a detailed description of all available functions.
MusicBrainz IDs An introduction to MusicBrainz identifiers.
TunePimp Docs The TunePimp documentation page explains how to use the TunePimp Library to create your own MusicBrainz enabled tagging application.
Metadata tags Lists all metadata tags that TunePimp (and therefore Picard and the Windows Tagger) write to music files.
CD submission overview This article gives a detailed explanation for some of the mysteries behind Disc IDs and why they behave they way they do.
Disc IDs A detailed article that describes how MusicBrainz calculates Disc IDs.
TRM Generation Describes how TRM generation works.
Tagger Traffic Dissecting a Tagger session. Protocol analysis.
Server/Database documentation Extensive description of our current database structure and the MusicBrainz server (mb_server).

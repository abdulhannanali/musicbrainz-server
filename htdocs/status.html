This page should give you a good overview of there the various portions of this project stand.

<& /comp/section, title=>'MusicBrainz Server' &>

Complete/Working Features

  • Support for storing and moderating the basic metadata (artist, album, title, track number)
  • Searching/browsing the database
  • Daily data dumps made available to the public.
  • Support for adding CDs via the web interface
  • RDF web services API to search/retrieve data from MusicBrainz.
  • Artist alias support
Needed Features before a 1.0 Release

  • Database commit/rollback support
  • Database replication using MySQL
  • Support for Merging/Deleting complete Albums
  • Add support for submitting tracks/CDs via the RDF interface. These submissions must then be passed through the moderation system.
<& /comp/section, title=>'MusicBrainz Client Library' &>

Complete/Working Features

  • Complete/working reference implementation.
  • Support for looking up Audio CDs.
  • Support for looking up/searching metadata.
  • Support for generating TRM Ids from audio data.
